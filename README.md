# Armoury Child Theme

A professional WordPress child theme for Kadence, crafted by Armoury Media.

## Requirements

- WordPress 6.0+
- Kadence Theme
- Kadence Blocks (recommended)
- Kadence Blocks Pro (recommended)

## Features

- Custom login page styling
- Bunny Stream video embed support
- Clean, maintainable code
- Performance optimized

## Installation

1. Install and activate Kadence theme
2. Upload this child theme
3. Activate the child theme

## Usage

### Video Embeds
Link any image to a Bunny Stream video URL:
```html
<a href="https://iframe.mediadelivery.net/play/[video-id]">
    <img src="thumbnail.jpg" />
