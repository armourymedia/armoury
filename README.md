# Armoury Child Theme

A WordPress child theme for Kadence, crafted with care by Armoury Media.

## Requirements

* WordPress 6.0+
* Kadence Theme (parent)

## Installation

1. Upload to `/wp-content/themes/armoury/`
2. Activate via Appearance > Themes
